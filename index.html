Stack is a presentation library
based on Mike Bostock's stack.js.

I have modified it to behave more like traditional presentation software.

Links:

You can change sections with the keyboard:
use the arrow keys,
the space bar,
or page up and page down
or home and end
or the mouse wheel

Multiple key presses will skip multiple sections.

= key opens up navigation mode

animations are a bit buggy still

You can even grab the scroll bar!

Even though nothing will change!
